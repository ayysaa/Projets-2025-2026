Tous les projets réalisés sont issus d'un travail de groupe assidu. Ainsi, certaines citations ou parties de code ne proviennent pas de moi.
Les collbaborateurs de ces projets sont Anthony Millerioux, Tim Salva et Noa Gillet.
La composition du groupe de travail a pu varier, mais les noms de mes collbaborateurs sont présents sur les rapports de chaque projet.

Durant mon année universitaire en BUT Informatique, j'ai réalisé les projets suivants :

- Projet C (début d'année)
(dossier : projet_C)
Le projet C est un jeu du Taquin. Le principe est de remettre en ordre une image mélangée et coupée en morceaux en cliquant sur un morceau qui se déplacera vers la case vide.
Le résultat est accessible via le fichier "main.c". Cependant, il faudra au préalable installer la bibliothèque graphique dédiée de l'université, disponible à ce lien :
https://iut-fbleau.fr/sitebp/doc/doc_bib_graphique/index.php?p=1
Le fichier "rapport.pdf" détaille l'ensemble du fonctionnement et de la réalisation du projet, et contient quelques captures d'écran du résultat.

- Projet WEB (milieu d'année)
(dossier : projet_WEB)
Le projet WEB a commencé par la création d'une entreprise dans le cadre du projet d'économie, suivie du développement du site pour le projet web.
Le résultat est accessible en ouvrant le fichier "index.html" dans un navigateur.
Le fichier "Rapport_Javascript_GUINET_GILLET_SALVIA" est un rapport centré uniquement sur l'utilisation du JavaScript dans notre site. (fichier : projet_WEB)

- Projet BD (toute l'année)
(dossier : projet_BD)
Le projet BD, réalisé sous Oracle en SQL, se divise en trois parties : la première porte sur la modélisation avec StarUML, la deuxième sur la création de la base de données, et la dernière sur sa phase de test.
Le résultat est visible à travers les nombreuses captures d'écran intégrées aux trois fichiers PDF de rapport, qui correspondent au rendu final.

- Projet PHP (fin d'année)
(dossier : projet_PHP)
Le projet PHP consiste à concevoir un site internet servant de bibliothèque de jeux vidéo.
Le résultat est accessible via ce lien :
https://dwarves.iut-fbleau.fr/~millerio/sae_r301_gr12/
Le fichier "JOURNAL.md", situé dans le dossier "projet_php", recense l'évolution de notre site.

Projet JAVA (fin d'année)
(dossier : projet_JAVA)
Le projet JAVA est un jeu du démineur classique.
Le résultat est accessible simplement en lançant le fichier "Display.java" dans le répertoire : <projet_JAVA> -> <src> -> <main>, ou en consultant le rapport qui contient des explications détaillées et des captures d'écran.
Le fichier "rapport.pdf" est un compte rendu complet qui présente l'ensemble du fonctionnement et de la réalisation du projet.

Projet SCR (toute l'année)
(dossier : projet_SCR)
Le projet SCR est un projet en cinq parties sur une machine virtuelle (VM), dont le but était d'installer un Dual Boot Linux & Windows, puis de configurer un réseau fonctionnel.
Le résultat n'est pas directement accessible, car il se trouve sur une VM fournie par l'université, uniquement accessible via mes identifiants et ceux de mon binôme. Le rendu final est toutefois conforme aux consignes.
Le fichier "guinetd-millerio-26.pdf" est le seul rapport disponible ; il a été rédigé à l'issue de la deuxième partie du projet (sur cinq au total). Les autres fichiers PDF correspondent aux consignes.
